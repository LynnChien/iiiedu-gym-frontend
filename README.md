## 資策會MFEE07大專題

### 第三組 健身房官方網站

+ 前端技術：html5 + sass + react.js + redux + javaScript
+ 後端技術：node.js + express.js + mysql
+ 功能說明：
  + 會員系統（CRUD）
  + Google登入
  + 商城（包含購物車、願望清單、訂單管理）
  + 課程預約系統（CRU）
  + 教練管理課程（CRUD）
  + 線上討論區（CRUD）
  + 客服中心（CRU）

### 如何使用
+ 安裝/yarn install
+ 啟動/yarn start
